# curiosity-board

**个人好奇心看板 —— 把值得探索的问题先存起来，等有空再慢慢研究**

---

### 项目初衷

我们每天都会冒出很多有趣、有价值、值得深入思考的问题，但往往因为当下没时间，就让它们白白溜走了。

**CuriosityBoard** 就是为了解决这个问题而生的：

> **先把问题安全地「存」下来，再找合适的时间「挖」深挖透。**

它不是简单的问题清单，而是一个**个人知识管理 + 生产力看板**，帮助你系统化地收集、管理和探索自己的好奇心。

---

### 核心理念

- **收集阶段**：想到就记，降低心理负担
- **管理阶段**：像任务看板一样分类、优先级排序
- **探索阶段**：有空时逐个攻克，转化为自己的知识
